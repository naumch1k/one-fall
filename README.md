## Installation & Usage

### Local development
To install the application:

```bash
git clone https://github.com/naumch1k/one-fall.git
cd one-fall
yarn install
```

To start a local copy of the app on port [http://localhost:8888](http://localhost:8888):

```bash
yarn dev
```
